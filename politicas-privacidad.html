Información que Recopilamos

Información Voluntaria

Solo recopilo información que decides compartir voluntariamente:

  • Información de contacto cuando me envías un mensaje

Información Automática

Datos técnicos básicos que se recopilan automáticamente:

  • Dirección IP (anonimizada)
  • Tipo de navegador y dispositivo

Importante: No utilizo herramientas de tracking invasivas como Google Analytics. Solo recopilo información estadística básica necesaria para el funcionamiento del sitio.

Seguridad de los Datos

Implemento las siguientes medidas de seguridad para proteger tu información:

Encriptación HTTPS

Todo el tráfico entre tu navegador y el sitio está encriptado con SSL/TLS.

Hosting Seguro

Utilizo Vercel, una plataforma confiable con altos estándares de seguridad.

Minimización de Datos

Solo recopilo la información absolutamente necesaria para el propósito específico.

Retención Limitada

Los datos se mantienen solo el tiempo necesario y se eliminan de forma segura.

Transparencia: Aunque implemento medidas de seguridad robustas, ningún sistema en internet es 100% seguro. En caso de cualquier brecha de seguridad que pueda afectar tus datos, te notificaré inmediatamente.
